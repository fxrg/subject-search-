### 💻 **للمطورين - تشغيل محلي**

```bash
# 📦 استنساخ المستودع
git clone https://github.com/fxrg/subject-search.git

# 📁 الانتقال للمجلد
cd subject-search

# 🌐 تشغيل الخادم المحلي
python -m http.server 8000
# أو
npx serve .

# 🎉 افتح المتصفح على localhost:8000
```

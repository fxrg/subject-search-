سياسة الخصوصية

نقدّر خصوصيتك ونلتزم بحماية المعلومات التي تشاركها معنا. توضح هذه الصفحة كيف نجمع ونستخدم ونخزّن المعلومات في مشروع subject-search.

الخدمات المستخدمة

يستخدم هذا المشروع خدمات Firebase (Authentication و Cloud Firestore و Storage) كبنية خلفية لمصادقة المستخدمين وتخزين البيانات والمحتوى الذي يساهم به المستخدمون. بياناتك قد تُخزن في قواعد بيانات Firebase كما هو مطلوب لتوفير ميزات الموقع.

البيانات التي نجمعها

  • بيانات الحساب: بريد إلكتروني واسم الحساب عند التسجيل عبر Firebase Authentication.
  • محتوى المواد: العناوين، الوصف، الروابط، والموارد التي يساهم بها المستخدمون.
  • بيانات تقنية: متعلقات الجهاز، السجلات، وبيانات التصفح الأساسية المستخدمة لتحسين الأداء والتشخيص.
  • إعدادات محلية: تفضيلات العرض (الوضع الداكن/الفاتح)، لغة العرض، وبيانات محفوظة في localStorage أو ملفات تعريف الارتباط لضبط تجربة المستخدم.

كيف نستخدم البيانات

نستخدم البيانات لتشغيل وظائف الموقع (مثل البحث، عرض النتائج، والمشاركة)، لتحسين جودة الخدمة، وللتواصل مع المستخدمين عند الحاجة (إشعارات إدارية أو استعادة حساب).

الاحتفاظ بالبيانات وحذفها

يمكن للمستخدمين طلب حذف بياناتهم عبر التواصل مع فريق المشروع (أو عبر واجهات إدارة الحساب عندما تكون متاحة). سنستجيب لطلبات الحذف وفقًا لإمكانياتنا ومع مراعاة متطلبات النظام.

الأمان

نستخدم ممارسات أمان قياسية عند التعامل مع البيانات ونستفيد من ميزات أمان Firebase. رغم ذلك، لا يوجد نظام يمكن أن يكون آمنًا بنسبة 100%، وننصح المستخدمين بتوخي الحذر عند مشاركة معلومات حساسة.

تغييرات سياسة الخصوصية

قد نحدّث هذه السياسة من حين لآخر. سيُعلن عن التغييرات عبر الموقع أو عبر وسائل الاتصال المتاحة.

تواصل

لأي استفسارات حول الخصوصية أو طلبات حذف البيانات، يمكنك فتح Issue في المستودع أو التواصل عبر وسيلة الاتصال المتاحة في المشروع.
